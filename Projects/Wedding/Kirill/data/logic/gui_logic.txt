Gui -> кнопка которая перемещается. Если она была нажата один раз, то она перестает перемещаться. Таймер запускается в первый раз для быстрого отсчета времени, чтобы никто не успел прочитать ни условий ни отправить сообщения в чат! Запускается второй раз запускается с нормальным таймером и сбором информации. После запускается прогонка скрипта.(Сделать вывод информации, придумываем слова, подбираем звук, накладываем голос, записываем песню)
в процессе выполнения скрипта создаю Label на все окно приложения, которое отображает статус. По завершению открываю фотографию, которую сгенерировала нейросеть вместе с песней.
