В файле хранятся информация о массе предмета и его обьеме, количество данных не указано
Разработать программу , которая будет считывать из файла данные по ключам
На вызов программы:
		С ключом :
			ALL :
				-Выводить информацию о всех предметах в файле
			Любой :
				-Выводить список данных с указанной подстрокой
		Без ключа :
			- Выводить упорядоченный список в от минимальной плотности к максимальной
Функци работы программы:
	Работа со структурой:
		1) Выделение памяти ( Создание ) структуры для предмета
		2) Освобождение памяти из-под структуры 
 	Работа с файлом:
		1) Проверка файла на корректность
	Массив структур:
		1) Создание массива структур
		2) Освобождение памяти из-под массива структур
		3) Вывод на экран массива структур
	
	Основные функции 
		1) Сортировка по возрастанию
		2) Поиск по подстроке

Декомпозиция на 3 заголовочных файла:
		1) Arr.h --работа с массивом структур
			- Созданеи массива структур
			- Освобождение места из-под массива структур
		
		2) Basic.h -- основные функции программы 
			- Поиск по подстроке
			- Сортировка по возрастанию

		
		3) Io.h   --работа с вводом выводом
			- Вывод массива структур
			- Вывод элемента массива структур
