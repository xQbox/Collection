Базовое задание для программы:
	 Оформить программу, которая сравнивает скорость умножения разреженных матриц,
	 сохраненных в виде:	
			1) Стандратное представление
			2) Массив значений + массив позиций + массив указателей на позиции

Декомпозиция программы:
	Ввод/ Вывод :
		1) Ввод матриц 
			1.1) Размерность
				- Больше 5	
					Ввод поэтапно i j элемент
	
				- Меньше 5
					Ввод по индексам (сначала задается позиция элемента)

			1.2) Случайня матрица
		2) Вывод результатов (полученная матрица) 
			2.1) Обычный вид матрицы
			2.2) Массив значений + ...
		
		3) Перевод в "Массив значений"
			

	Основная программа:
		1) Cоздание структуры данных (разреженная матрица -)
				- Массив значений
				- Массив элементов по столбцам
				- Массив количества элементов по строкам
